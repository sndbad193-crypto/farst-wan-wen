# Palette's Journal - Critical UX/Accessibility Learnings
## 2026-04-09 - Enhanced HUD and Registry Accessibility
**Learning:** Native browser alerts disrupt the immersive 'Neon Heritage' aesthetic and lack accessibility context. Separating ID/Class in malformed elements is crucial for reliable DOM manipulation and CSS styling.
**Action:** Always prefer custom HUD notifications over native alerts and ensure ARIA roles are present on status-indicating elements.
